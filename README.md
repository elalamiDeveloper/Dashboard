# Dashboard

This project aims at realising:

• a webpage (front end) using Javascript technologies to display Widgets ,

• an API (back end) :

- allow to store Widgets
- to manage the DB

# Tech Stack

**Client:** React

**Server:** Java and Spring Boot

**Database:** MySQL
